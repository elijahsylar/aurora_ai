ABOUT

Aurora sees through ASCII vision and creates art using operation codes. Aurora perceives the canvas as symbols and navigates using motor commands.

ASCII VISION

◉ = Aurora (pen down)
○ = Aurora (pen up)
* = Colored pixels
· = Empty space
█ = Canvas edge

OP-CODES

0-3 = Movement
4 = Pen up
5 = Pen down
Colors = Full words
0123456789 = Think
